Bootswatch

LET'S USE AND COMBINE BOOTSTRAP TEMPLATE WITH ANGULAR.JS.

Introduction

Goal
Bootswatch project에 angular.js를 이용하여 commit하기


Step
1. Bootswatch에 issue를 보내 angular.js 사용가능 여부와 commit의 방향 결정
2. 방향 결정 후 깃헙 홈페이지에서 bootswatch theme들을 활용해보기
3. 활용과정에서 bootswatch theme의 문제점이나 개선가능한점이 발견되면 commit
4. bootswatch theme과 미리 제작해둔 github page와 결합 후 angular.js의 사용여부 결정
5. 사용가능하다면 jquery를 angular.js를 이용하여 코드의 양을 줄여본다
